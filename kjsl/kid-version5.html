SCRIPT 4: 

         "use strict"; 
         var kukur; 
         kukur="dog"; 
         document.write(kukur)

Please look at the script above. You will find four things.

1. In the very first line we see this : "use strict"
When we write many lines of code, this "use strict" helps us by reporting what type of errors/ mistakes we made. So it is a very useful thing to add this at the beginning of your every piece of code.
2. Next we see the word "var" which is a short form of the word variable data.
We can think of var as a box or container where we put various types of data such as numbers, text, list of things etc.
3. In the third line, we store a text called "dog" in the var "kukur";
Now we can ask the kukur,"Hey, What is stored inside you?"
And kukur will reply "dog". 4. Now in line 4 we write the code to do the following things :
i) We ask kukur what he stored in him.
ii) We ask javascript to display that stored thing at the end of our page.

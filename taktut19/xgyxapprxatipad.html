ङ्याप्प्रातिपदिकात् ONPANINI 41001

After /GI, /Ap and nounbases, (some /sup is always added).

The list of all /sup is svaujasamauTch... >. Other sUtras explain which one must be added in each case.

Example. After the nounbase azva- अश्वॱ, sometimes you have to add /su, which is a /sup. The result is the noun azvas अश्वस् —

azva- अश्वॱ + /su → !**azvas अश्वस्

rAma- रामॱ + /su → !**rAmas रामस्

guru- गुरुॱ + /su → **gurus गुरुस्

In other situations you have to add /am, or any of the other twenty /sup —

azva- अश्वॱ + /am → !**azvam अश्वम्

A nounbase is a theoretical construct as is only found in dictionaries and grammars. To get an actual noun that can be used in a sentence, you always have to add some /sup or other.

But, in some website, I read hasti azva ratha pAdAtam divya rUpam svalaGkRtam . Sure azva appears without any /sup ending there?

Indeed it does. This rule says that some rule HAD to add a /sup after that azva. If you can hear no /sup there, that's because some rule added one, then some other rule removed it later. In this case, rule supodhAtupr... removed it.

And BTW that line should be spelled as hastyazvarathapAdAtan divyarUpaM svalaGkRtam . You cannot trust that website.

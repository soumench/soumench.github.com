Javascript is a very popular programming language for the web. It is good if you know it. You can create many funny and interesting things with it.

A script tag is the place where we put our code so that the browser understands where the javascript is written. It is like this : 

 < script > ................. < /script > </pre >

Let's start with a very simple line of code.
SCRIPT 1: 

  document.write("dog")

If we write it correctly inside the script tag , we will know if javascript is listening to us or not. If we do it correctly, the script will output the word "dog". See the full code of an html page below: 

            < html >
            < head >   < /head>
            < body >
            < script >  document.write("dog") </script >
            < /body >
            < /html >

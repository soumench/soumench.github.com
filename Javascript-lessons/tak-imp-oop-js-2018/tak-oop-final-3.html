Lesson 3

How do we tell JS to create an object?

The following line does two things. first oit creates an object with no drawers.
Then it gives to it the name b2e.

var b2e={};



How do we tell JS to create a drawer with the key "kukkur" and the value "dog"?

we can always do this:

b2e["kukkur]="dog";

but it your key starts with a letter, and has only letters and numbers afterwards, you can also type:

b2e.kukkur="dog"



How can we ask JS what is the contents of the drawer kukur in the chest b2e?

When we want to send the value to the console, we do this:

console.log(b2e["kukkur"])

When we want to add the value to the end of the HTML page, we do this:

document.write(b2e["kukkur"])



How can we make an object have two names?

Do this: 

    var e2b={}; 
    var englishtobengali=e2b; 
    e2b.kukkur="dog"

After you do that it does not matter if you type e2b.kukur or bengali2english.kukur, you get to the same drawer anyway, and you get "dog".

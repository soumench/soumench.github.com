Lesson 1

1. What is an 'Object'?

In a computer program, an 'object' is something in which we store some data.



2. How do we store data inside an 'object'?

When we put data in the 'object', we have to say a "key", and the data.
And to recover the data, we have to give the key.

We can compare an 'object' with a furniture with many lockers. Each locker has a unique key to access what is inside.

Let's see that we want an 'object' to keep in it English translations of Bengali words.
We can give that 'object' a name, or several names.
For instance, it might be called bengalienglishdictionary, or b2e.
Then you have a way of saving the data "dog" under the key "kukur" into that 'object'.



3. how do we recover the data we saved inside an 'object'?

Whenever we ask the 'object' for the contents of the key kukkur, it will give us a fresh copy of that.
And if you ask again, you get the same thing again, the 'object' never forgets.



4. What if we ask for the contents of a drawer but we did not put it inside the 'object' earlier?

The 'object' gives us a thing called 'undefined'.

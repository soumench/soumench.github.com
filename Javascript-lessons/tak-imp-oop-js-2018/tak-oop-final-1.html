1. What is an 'Object'?

In a computer program, an 'object' is something in which we store some data.



2. How do we store data inside an 'object'?

When we put data in the 'object', we have to say a "key", and the data.
And to recover the data, we have to give the key.
We can compare an 'object' with a furniture with many lockers. Each drawer has a label and all the labels are different.

Let's say we want an 'object' and then we want to put the English translations of some Bengali Words in it.
We tell the furniture, please put the word "dog" into the drawer that has label "kukur". After a while we ask the furniture, read to me what you have in the drawer 'kukur' and the furniture replies "dog".
Now "dog" is said to be a value, and the "kukur" that we use to retrieve the value is said to be a "key".
